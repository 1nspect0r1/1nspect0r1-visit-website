ImgViewer

ImgViewer - Это приложение, разработанное для Windows, позволяет удобно просматривать изображения в реальном времени с поддержкой широкого спектра графических форматов.

Программа работает с большинством популярных графических форматов, включая:

  • JPEG/JPG
  • PNG
  • BMP
  • GIF (включая анимацию)
  • TIFF
  • SVG
  • WMF/EMF (метафайлы Windows)
C# .NetFramework Microsoft Windows Forms

OpenSpaceMonitor

OpenSpaceMonitor - это утилита для мониторинга системных ресурсов вашего компьютера с возможностью отображения текущей температуры воздуха в выбранном городе.

Основные возможности:

  • Мониторинг загрузки CPU и GPU
  • Отслеживание температуры компонентов
  • Отображение прогноза погоды
  • Настройка пороговых значений для уведомлений
Python 3.6+ requests psutil GPUtil

PasswordGenerator

PasswordGenerator - это простой генератор паролей с графическим интерфейсом, созданный для безопасного и удобного создания сложных паролей.

Основные возможности:

  • Настройка длины пароля
  • Выбор типов символов (буквы, цифры, спецсимволы)
  • Автоматическое копирование в буфер обмена
  • Оценка сложности пароля
